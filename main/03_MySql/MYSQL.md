##数据库的相关概念和特点

###数据库的好处

	1、永久存储数据
		文件
		数据库

	2、比较方便管理和查找
	3、安全性较高
	4、数据的共享性


###数据库的特点
	1、数据先存到表中，然后再将表存到库中
	2、格式相同的数据才能存放到一张表

	stuinfo学生信息表

		1  john	女	xx
		

	3、插入数据，每次插入都是一行
	   删除数据，每次删除都是一行

##数据库服务器和客户端的安装

方式一:
	右击——计算机管理——服务——mysql服务名
方式二：
	命令行方式

##sql语言概述
###sql语言的分类
	DML: Data Manipulation Language 数据操纵语言
	DDL:  Data Definition Language 数据定义语言
	DCL:  Data Control Language 数据控制语言

###sql语言的特点

	1.sql语言不区分大小，但建议关键字大写，表名、列名小写
	2.每一句话用分号
	3.按行或按句执行，使用快捷键F9
	4.mysql的注释：
	
	#   单行注释的文字
	--  单行注释的文字
	/**/多行注释的文字
